# Taskana App

**Продвинутый список задач (учебный проект)**  
Приложение для управления задачами с сохранением данных в браузере. Позволяет создавать, редактировать, фильтровать, сортировать задачи, а также переключать тему оформления. Сделано на React + Vite.

![Скриншот интерфейса] `[ДОБАВИТЬ_СКРИНШОТ_ИЛИ_ГИФКУ]`

## 🧩 Особенности

- ✅ Создание, редактирование и удаление задач
- 🔄 **Сортировка и фильтрация**:
  - по дате создания
  - по дате последнего редактирования
  - по приоритету (от большего к меньшему / от меньшего к большему)
- 🌓 **Светлая и тёмная тема** — переключается одним кликом
- 💾 Автосохранение в `localStorage` — все настройки и задачи сохраняются после перезагрузки
- ⏳ Имитация загрузки: при клике на кнопку добавлена искусственная задержка (чтобы показать `loading`-состояние)
- 📱 Адаптивная вёрстка (на узких экранах возможен горизонтальный скролл — см. раздел «Известные проблемы»)

## 🛠 Технологии

- **React** — библиотека для UI
- **Vite** — сборка и дев-сервер
- **JavaScript (ES6+)**
- **HTML5 / CSS3** (поддержка светлой/тёмной темы)
- **LocalStorage** — хранение данных и выбранной темы

## 🚀 Быстрый старт

### Требования
- Node.js (рекомендуется версия 18.x или выше)
- npm (или yarn/pnpm)

### Установка и запуск

```bash
# 1. Клонируй репозиторий
git clone https://github.com/твой-ник/taskana.git

# 2. Перейди в папку проекта
cd taskana

# 3. Установи зависимости
npm install

# 4. Запусти дев-сервер
npm run dev
